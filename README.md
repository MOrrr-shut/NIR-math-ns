# NIR-math-ns
Студенческая работа: оценка прогноза линейной регрессии и элементарной НС для линейной функции 

Начная работа, сравнение прогноза классических математических моделей и нейроного метода в простом варинанте.
Для более удобного и интересного написания: изучение графичской библиотеки QT6::Charts и поптыка написания абстрактного класса для создания слайдов с анимацией. 

*анимация получилось очень корявая((((

Итог:  
1) Графичесое ПО на QT
2) Анимация плавная
3) Абстрактный класс выполняет свой функционал
4) Линенйная регресия и нс с relu написана по учебнику
5) Демонстрация ПО на таблице изменения стоимости биткоина
